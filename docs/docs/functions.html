函数

fn add(x int, y int) int {  
    return x + y  
}  

fn sub(x, y int) int {  
    return x - y  
}  

fn main() {  
    println(add(77, 33))  
    println(sub(100, 50))  
}

同样，类型出现在参数的名称之后。

就像在Go和C中一样，函数不能重载。 这简化了代码并提高了可维护性和可读性。
